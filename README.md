# Portfolio Josue Adigoun - Version modifiable V2

## Nouveautés

Cette version permet de :

- modifier les anciens projets ;
- changer le texte des anciens projets ;
- ajouter ou changer l’image d’un ancien projet ;
- ajouter une image de fond ;
- ajouter de nouveaux projets ;
- supprimer des projets.

## Pages

- `index.html` : portfolio visible
- `admin.html` : page de modification

## Utilisation

1. Ouvre `admin.html`
2. Clique sur `Modifier` au niveau d’un projet
3. Change le texte ou ajoute une image
4. Clique sur `Enregistrer les modifications`
5. Ouvre `index.html` pour voir le résultat
